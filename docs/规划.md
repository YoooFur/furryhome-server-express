## 主要功能

- 直接返回全部列表

```json
// GET /site/list

{
  "code": 200,
  "msg": null,
  "data": {
    "data": [
      {
        "cateId": 1,
        "cateName": "信息平台",
        "cateIcon": "https://xxx.xxx/xxx.ico",
        "cateIntro": "这也是一条简介",
        "siteList": [
          {
            "siteId": 1,
            "siteName": "FurDevsCN",
            "siteIntro": "这是一条简介",
            "siteFavicon": "https://xxx.xxx/xxx.ico",
            "siteUrl": "https://xxx.xxx",
            "siteParam": "from=furryhome",
            "siteViews": 666,
            "siteLikes": 6,
            "siteCreateTime": 1651651177447
          },
          ...
        ]
      },
      ...
    ]
  }
}
```

- 返回类型列表

```json
// GET /cate/list

{
  "code": 200,
  "msg": null,
  "data": [
  {
    "cateId": 1,
    "cateName": "信息平台",
    "cateIcon": "https://xxx.xxx/xxx.ico",
    "cateIntro": "这也是一条简介"
  },
  ...
  ]
}
```
